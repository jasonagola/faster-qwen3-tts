# Text-Delta Streaming Samples

These samples compare the same text through text-delta input streaming and full-text audio streaming. Each pair uses the same prompt and generation settings except for the input path. They are generated by `benchmarks/text_delta_readme_benchmark.py` and validated for nonzero duration, finite samples, and non-silent peak amplitude.

Default sample settings: `chunk_size=8`, `token_holdback=1`, `do_sample=True`, `temperature=0.9`, `top_k=50`, `top_p=1.0`, and `repetition_penalty=1.05`.

| Mode | Text-delta input | Full-text input |
|---|---|---|
| CustomVoice 200-token sample | `custom_voice_200_text_delta.wav` | `custom_voice_200_fulltext.wav` |
| VoiceDesign 100-token sample | `voice_design_100_text_delta.wav` | `voice_design_100_fulltext.wav` |
| Voice clone x-vector 100-token sample | `voice_clone_xvec_100_text_delta.wav` | `voice_clone_xvec_100_fulltext.wav` |
